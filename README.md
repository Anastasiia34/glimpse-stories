# Glimpse

Instagram-like stories experience built with SwiftUI.

## Features

- Infinite horizontal stories pagination
- Story groups with auto-progress
- Tap navigation between story items
- Swipe navigation between story groups
- Swipe down to dismiss
- Long press to pause stories and hide overlay
- Seen and liked stories persistence

## Tech Stack

- SwiftUI
- MVVM
- Swift Testing

## Requirements

- iOS 26+
- Xcode 26+
